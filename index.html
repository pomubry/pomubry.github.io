Video Trimmer

A script-based video trimmer using FFmpeg, mpv, and Node.js.

Trim one or more segments from a video that you like and merge each one at the end. All you need to do is to save the filename and all the timestamps which you would like to be trimmed inside a text file and let the program handle the rest.
